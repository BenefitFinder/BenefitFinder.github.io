# BenefitFinder.github.io
Repository for the BenefitFinder project of WWU course CSCI 436/536
Link to our website: https://benefitfinder.github.io/
If you wish to run our website locally you'll need to install ruby and jekyll
# Jekyll Documentation: 
https://jekyllrb.com/docs/installation/ (see the guides at the bottom for your system)

Once jekyll is installed you can run `gem install github-pages`
followed by `jekyll build` and `jekyll serve` which will serve the page locally to http://127.0.0.1:4000
